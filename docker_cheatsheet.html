Docker Commands Cheatsheet

1. Docker Basics

Command Description
docker --version Check Docker version.
docker info Display system-wide Docker information.
docker help List all Docker commands.

2. Docker Images

Command Description
docker images List all available images.
docker search <image> Search for an image on Docker Hub.
docker pull <image> Download an image from Docker Hub.
docker build -t <name>:<tag> . Build an image from a Dockerfile.
docker rmi <image> Remove an image.
docker image prune Remove unused images.
docker inspect <image> Display detailed information about an image.

3. Docker Containers

Command Description
docker ps List running containers.
docker ps -a List all containers (including stopped).
docker run <image> Run a container from an image.
docker run -it <image> Run a container interactively (e.g., with bash).
docker start <container> Start a stopped container.
docker stop <container> Stop a running container.
docker restart <container> Restart a container.
docker rm <container> Remove a container.
docker logs <container> View logs from a container.
docker exec -it <container> bash Access a running container's shell.
docker inspect <container> View container details.
